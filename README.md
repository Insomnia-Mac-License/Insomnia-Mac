## How to Install

1. Download the latest version of Insomnia for macOS.  
2. Open the DMG installer and move Insomnia.app to the Applications folder.  
3. Launch the app and allow it through macOS Gatekeeper if required.

## Overview

<a href="#">Insomnia for macOS</a> is aimed at engineers who want an API client that feels like part of their toolkit rather than a separate portal. It focuses on keeping requests, environments and authentication details organised in a simple tree so that everyday work—like reproducing a bug or testing a new endpoint—stays fast. Dark-theme friendly visuals and compact layouts make it comfortable during long coding sessions, especially when it sits side by side with an editor and log viewer.

The client supports REST, GraphQL and other paradigms, with body templates, variable substitution and <a href="#">chained requests</a> for more complex workflows. Environment management helps separate credentials and base URLs by stack, keeping sensitive information contained while still enabling quick switching. Plugins and scripting hooks allow power users to extend behaviour, integrating custom auth flows, signing logic or transformations that match their infrastructure. This makes Insomnia useful in both greenfield services and heterogeneous legacy environments.

On macOS, Insomnia takes full advantage of window management and keyboard navigation so users can keep it in the background until it is needed, then bring it forward for focused debugging. Requests can be cloned, grouped and annotated for future runs, turning the workspace into an evolving diary of how an API has behaved across versions. Over time, it becomes the trusted place to confirm what a service is really doing, independent of code expectations or third-party docs.

## Major Features
- Clean request tree for organising REST and GraphQL calls  
- Environment and variable support for multiple backends  
- Plugin and script hooks for custom auth and signing flows  
- Support for common authentication schemes and cookies  
- Powerful body editors for JSON, forms and raw payloads  
- Dark-mode friendly design tuned for long coding sessions  
- macOS keyboard shortcuts for quick navigation and sending  
- Response views with formatting, previews and timing data  
- Workspace separation for different projects and clients  
- Import compatibility with popular API definition formats  
- Sync and backup options to protect everyday work assets  
- Ideal as a focused companion to editors and log viewers
